Résumé

Ce projet explore l'efficacité de différents formats de compression d'image appliqués aux lames histopathologiques (Whole Slide Images — WSI).

Dans un premier temps, un benchmark systématique compare trois formats de compression classiques — JPEG, WebP et JPEG2000 — sur le dataset SVS-TCGA-2048. Les métriques PSNR et SSIM sont mesurées pour chaque format à différents niveaux de qualité (QF), permettant d'identifier le meilleur compromis entre fidélité visuelle et réduction de la taille de fichier.

Les résultats montrent que WebP offre le meilleur rapport qualité/taille pour les lames histopathologiques, suivi de JPEG2000 puis JPEG. La réduction de taille peut atteindre 97% tout en maintenant une qualité visuelle diagnostique acceptable (PSNR supérieur à 30 dB).

Dans un second temps, une expérimentation alternative est menée : la conversion des images vers le format vectoriel SVG via l'outil VTracer. L'objectif est d'évaluer si la représentation vectorielle peut offrir une compression efficace pour ce type d'images. Les résultats indiquent que le SVG est en moyenne 10× plus volumineux que le PNG original sur les images histopathologiques, en raison de leur nature riche en textures qui résiste à la vectorisation.

Méthodes de compression

Compression classique : JPEG, WebP, JPEG2000

JPEG est le standard historique de compression d'image, basé sur la DCT (Discrete Cosine Transform) avec quantification des coefficients. WebP, développé par Google, combine compression intra-frame (VP8) et prédiction pour offrir de meilleurs ratios. JPEG2000 utilise la transformée en ondelettes (DWT) pour une compression multi-résolution plus efficace, particulièrement adaptée aux grandes images comme les WSI.

Le benchmark mesure le PSNR et le SSIM pour chaque format à différents niveaux de qualité (QF = 10, 30, 50, 70, 90), sur le dataset SVS-TCGA-2048 (407 images de 2048×2048 pixels extraites de lames TCGA).




Approche vectorielle : SVG avec VTracer

VTracer convertit les images matricielles en SVG vectoriel pur (chemins <path>) en détectant automatiquement les contours et couleurs. L'objectif est d'évaluer si la représentation vectorielle peut compresser efficacement les lames histopathologiques.


Résultat SVG :

Sur les images histopathologiques riches en textures, le SVG est en moyenne 10× plus volumineux que le PNG original. La vectorisation n'est donc pas adaptée à ce type d'imagerie médicale, car la complexité texturale empêche une représentation économe en primitives vectorielles.

Artefacts de compression

Les différents formats de compression introduisent des artefacts visuels caractéristiques. JPEG produit des effets de blocs (blocking artifacts) et du flou autour des contours, tandis que WebP et JPEG2000 préservent mieux les détails fins grâce à leurs algorithmes plus sophistiqués. La détection de ces artefacts est cruciale pour l'analyse diagnostique.

Compression Neurale (CINR)

Une approche expérimentale de compression utilise les Représentations Neurales Implicites (CINR - Compression with Implicit Neural Representations). Contrairement aux codecs classiques, cette méthode entraîne un réseau de neurones (Fourier features + SineCNN) à reproduire l'image à partir de ses seuls poids. Le modèle est sur-ajusté (overfitting) sur une image spécifique, et les poids du réseau constituent alors la représentation compressée.

Cette approche, inspirée du papier de Lee et al. (MICCAI 2024), est particulièrement prometteuse pour les images médicales car elle peut apprendre les caractéristiques spécifiques du domaine histopathologique.

Résultats : Le modèle CINR avec ~1M paramètres atteint 33.8 dB sur un patch 256×256 après 5000 itérations (173s sur M4 Pro). Cependant, la comparaison avec JPEG montre que ce dernier reste supérieur en termes de ratio qualité/compression : JPEG Q=30 donne 35.9 dB à 4.1 MB (ratio 24:1), tandis que CINR 1M float16 donne ~29 dB à 2.1 MB (ratio 48:1).

Conclusion : Bien que les INR offrent une représentation continue (zoom sans pyramide), elles ne sont pas encore compétitives face à JPEG pour le stockage de WSI à grande échelle.
